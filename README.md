# Interview Scheduler

A single-page application for scheduling interviews. Built with React.

[Demo](https://scheduler-miguel.netlify.app/)

!["Screnshot ..."](docs/screenshot.png)

## Usage

Install dependencies:

```sh
npm install
```

Run server locally:

```sh
npm start
```

Run Storybook Visual Testbed:

```sh
npm run storybook
```
